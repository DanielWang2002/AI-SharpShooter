# FPSGame-AIPlugin
一個可使用在大多數FPS遊戲上的輔助程式，使用`mediapipe`製作

#### 已測試過的遊戲
- `Apex 英雄`
- `Counter-Strike: Global Offensive`

## 安裝依賴
```
Python Version == 3.8
pip install -r requirements.txt
```

## 執行程式
```
打開FPS遊戲
python3 main.py
```
